# How to Run the Programs

## Practical 3a - Histogram

```bash
flex histogram.l
gcc lex.yy.c -o histogram -ll
./histogram
```

## Practical 3b - Take Input and Print Lines, Chars, Spaces

Create required files:
```bash
touch input.txt
touch output.txt
```

Run the program:
```bash
flex practical3.l
gcc lex.yy.c -o pr3 -ll
./pr3
```
